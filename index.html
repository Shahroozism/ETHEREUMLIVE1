Dapp Development

Ethereum is a platform that is intended to allow people to easily write decentralized applications (Đapps) using blockchain technology. Rather than serving as a front-end for selling or providing a specific party's services, a Đapp is a tool for people and organizations on different sides of an interaction used to come together without any centralized intermediary.

Smart Contracts

A smart contract is a computer protocol intended to digitally facilitate, verify, or enforce the negotiation or performance of a contract. Smart contracts allow the performance of credible transactions without third parties.

Learn How to Query & Monitor Ethereum Contracts

With Web3.js, you can query and listen for smart contract events on the Ethereum blockchain so that you can specify actions to trigger when certain criteria are met. The smart contract example we develop in this blog post use the Infura API, Etherscan API, and Wrapped ETH. Infura is a hosted Ethereum node cluster that lets users run their application without requiring them to set up their own Ethereum node or wallet. You may not be familiar with Infura by name, but if you've used MetaMask then you've probably have used Infura, as it is the Ethereum provider that powers MetaMask. We'll then find an example smart contract address on Etherscan, a API that lets you explore and search the Ethereum blockchain for transactions, addresses, tokens, prices and other activities. Finally, we apply the Wrapped ETH because it has a high volume of transcactions and makes it a great point of reference to use for texting event listening.

Making a Event Query

There are two primary use cases for interacting with Ethereum contract events. You can: you can monitor and query a smart contract's events or a user can create an event lisener for a smart contract's events to specify particular actions you want to trigger.

Query Past Events with getPastEvents()

Use Web3 to query all past events that a smart contract has emitted. The getPastEvents() will return the events all-in-one-shot.
